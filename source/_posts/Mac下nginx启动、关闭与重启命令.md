记录一下经常使用的nginx命令。<!--more-->
#### nginx相关命令
<!-- more -->
##### 1、启动nginx：

```
	sudo nginx
```
##### 2、关闭nginx：

```
	sudo nginx -s  stop
```
##### 3、重启nginx：
```
	sudo nginx -s reload
```
